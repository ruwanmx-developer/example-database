# example-database

<i><b>This is an example java project that uses the basic codings for manage a database with 'mysql-connector-java-5.1.48-bin' connector.</b></i>

<b>Basic Functionalaties</b>
  1. Create Table
  2. Delete Table
  3. Add Data
  4. Remove Data
  5. Update Data
  6. Refresh Data

<b>Other Features</b>
  1. Validate Data
  2. Enable / Disable UI Elements
